# MediAssist AI — Frontend

A premium React (Vite) frontend for the MediAssist AI product.

Quick start:

```bash
npm install
npm run dev
```

Environment:
- Create a `.env` file with `VITE_API_BASE_URL` pointing at your API.

Tech:
- React + Vite
- MUI v5
- Axios
- React Router

This repo contains a polished dashboard, upload flow, chat assistant, and history table.
